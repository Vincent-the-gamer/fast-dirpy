A simple CLI to download video using `dirpy.com`

> [!NOTE]
Work in progress.

## Installation

> [!CAUTION]
Not released yet.

```shell
npm i fast-dirpy -g
```


## Usage
```shell
# get video direct link
# Proxy:
#   -H, --proxyHost  host
#   -P, --proxyPort  port
fast-dirpy get http://xxx/yyy -H 127.0.0.1 -P 7890
```
